--Types Of Network--

En Yaygın Kullanılan
-Local Area Network (LAN) --> Aynı Locasyonda Kurulan , Kullanılan Ağ.
-Wide Ares Network (WAN) --> Birbirinden Uzak Lokasyonda Olan LAN'ları birbirine bağlayan ağa denir

Bazı diğer ağ türleri
-Metropolitan Area Network (MAN) --> Wide Ares Network'un aynı şehirde olan versiyonudur , yinede çok kullanılmaz aynı şehirde olsa bile WAN denir ama ek bilgi bilsen iyi olur.
-Wireless LAN (WLAN)
-Storage Area Network (SAN)

-Internet --> Dünyada Kullandığımız erişebildiğimiz ağ
-Intranet --> Sirket Tabanlı Kapalı Ağlar Örnek Olabilir
-Extranet --> Farklı Şirketlerin Ağlarını Birleştirdiği Yapı

C I A --> Dijital Alt Ypının Güvenli Kabul Edilebilmesi İçi 3 şart ; 
Confidentiality (Gizlilik) --> Dosyayı Sadece Senin izin verdiklerin görebilmeli veyea okuyabilmeli
Integrity (Bütünlük) --> Bir Dosya Sana Aitse onu sadece sen değiştirebilirsin ve kimse değiştiremeyecek sekilde ayarlanması lazım
Availability(Erişilebilirlik) --> İhtiaycın Olan veriye ihtiayc anında ulaşabilme

---OSI Modeli---   
L7 - Application --> Kullanıcı arayüzü    \  
L6 - Presentation --> İletim bicimi datayı  |  --> 7,6,5 katmanda datamız oluşuyor
L5 - Sessions --> oturum kontrolu         /
L4 - Transport --> paketleri segmentlere ayırıp numaralandırıyor ve kaynak ve hedef port numaraları ekleniyor
L3 - Network --> kaynak ve hedef ip numaraları ekleniyor
L2 - Data Link --> kaynak ve hedef mac addresslerim ve hata kontrol ekleniyor
L1 - Physical -->oluşan paket bit değerleri olarak iletiliyor

---TCP/IP Model---
Application --> OSI Modelindeki L7, L6, L5 Katmanlarının işini yapar
Transport --> OSI Modelindeki L4 Katmanının işini yapar
Internet --> OSI Modelindeki L3 Katmanının işini yapar
Network Access --> OSI Modelindeki L2, L1 Katmanlarının işini yapar

L7 + L6 + L5 = DATA
L7 + L6 + L5 + L4 = SEGMENT
L7 + L6 + L5 + L4 + L3 = PACKET
L7 + L6 + L5 + L4 + L3 + L2 = FRAME
L7 + L6 + L5 + L4 + L3 + L2 + L1 = BİT
